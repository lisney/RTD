기어 
------

```
모듈m = 피치원지름pcd / 잇수z => pcd = mz
기어거리d = (pcd1 + pcd2)/2 = (m*z1 + m*z2)/2 = m(z1+z2)/2
d = m(z1+z2)/2 => m = 2d/(z1+z2)

```

sub-Shape binde(초록색)
-----------------------

```
Part WB에서 생성한 Cube에는 기본적으로 Shape Binder가 안 먹힌다

-면 추가 방법 > 면을 선택 후 Cube아이콘을 Binder 아이콘으로 드랙한다(Ctrl-Drag 하면 새로 선택한 면으로 리셋)
```
